# Knowledge Base

> Compound Writing System 的本地知识库。创作时 Claude 自动读取，无需手动复制粘贴。

---

## 目录结构

```
02-knowledge-base/
├── _inbox/              ← All-in-one Inbox 自动存档（文字/语音素材）
├── 本专业信息/           ← 你的个人信息 + 产品信息
└── 爆款素材/             ← 按行业分类的参考素材
    ├── _inbox/          ← 未分类素材暂存
    ├── AI/              ← AI / Agent / LLM
    ├── IP/              ← 创作者 IP 方法论
    ├── Web3/            ← Web3 / 区块链
    ├── 商业/             ← 商业模式 / 市场
    └── 心理学/           ← 传播心理学 / 认知偏差
```

---

## 使用方式

创作时直接说：
- 「在知识库找关于 Agent 的内容」
- 「找之前存的素材」
- 「有没有关于 Web3 安全的参考案例」

Claude 会自动搜索匹配的文件，提取相关内容用于创作。

---

## 素材文件格式

```markdown
# 标题
**来源**：[平台 + 链接]
**日期**：YYYY-MM-DD
**标签**：#tag1 #tag2

## 核心内容
[关键信息 / 原文摘录]

## 可用素材
[金句 / 案例 / 数据——创作时可直接引用的部分]

## 相关选题
[这个素材适合什么写作场景]
```

---

## 维护建议

- **每 2-4 周**：把 `_inbox/` 中的有价值素材整理到对应分类
- **发布后**：高效内容（点赞/书签超均值）加入爆款素材
- **定期更新**：行业知识保持时效性，删除过时内容
