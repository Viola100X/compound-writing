# 进度面板

每个阶段转换时展示此面板，让用户始终知道自己在哪。

## 格式

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Compound — {当前阶段名}
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
选题：{选题内容}
结构：{结构类型 / 待定}
文风：{已初始化 / 通用风格}

{阶段清单}
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 阶段清单符号

- `[x]` — 已完成（附简要结果）
- `[>]` — 当前进行中
- `[ ]` — 待完成
- `[-]` — 已跳过

## 完整清单

```
[ ] Personal OS    初始化         — {完成度}
[ ] 收集素材       inbox-capture  — {素材数量}
[ ] 选题评估       topic-engine   — {评分/50}
[ ] 选结构         content-struct — {结构类型}
[ ] 深化内容       journalist/升华 — {轮次}
[ ] 出初稿         writing-style  — {字数}
[ ] 标题/钩子      title-gen      — {方案数}
[ ] 发布复盘       content-review — {评分}
```

## 示例

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Compound — 深化内容
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
选题：为什么我们会被张雪打动
结构：C 型 故事叙事型
文风：通用风格（未初始化）

[-] Personal OS    初始化         — 跳过
[-] 收集素材       inbox-capture  — 跳过
[x] 选题评估       topic-engine   — 37/50 A级
[x] 选结构         content-struct — C 型 故事叙事型
[>] 深化内容       thought-elev   — 理论透镜进行中
[ ] 出初稿         writing-style
[ ] 标题/钩子      title-gen
[ ] 发布复盘       content-review
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

A) 继续 → 出初稿
B) 跳到 → 指定阶段
C) 暂停 → 下次继续
```

## 阶段导航（面板后展示）

每次展示面板后，附上导航选项：

```
A) 继续 → {下一阶段}: {简述}
B) 跳到 → 指定阶段编号
C) 暂停 → 保存进度，下次继续
```
