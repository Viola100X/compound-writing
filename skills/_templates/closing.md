# 结束面板

当用户完成一篇文章的完整流程（或中途说「结束」）时，展示此面板。

## 三段式结束

### 第一段：创作回顾

```
本次创作回顾

- 关键决策：{回顾过程中最影响文章质量的一个决定，引用具体内容}
- 最大提升：{哪个阶段对文章改进最大，具体说明}
- 下次可复用：{一个可带走的经验，适用于未来写作}
```

### 第二段：数据统计

```
| 指标 | 数值 |
|------|------|
| 总耗时 | {分钟} |
| 使用 Skills | {列出用了哪些} |
| 文章字数 | {字数} |
| 选题评分 | {分数}/50 |
| 发布平台 | {平台} |
```

### 第三段：下一步推荐

```
接下来可以做：

A) 「帮我做发布复盘」— 发布后收集数据，分析什么有效
B) 「帮我找选题」— 基于这次经验，找下一个选题
C) 「建立文风」— 用这篇文章作为语料，初始化文风 Skill
```

## 系统进化检查

结束时自动检查：
- 本次改稿中有没有出现 3 次以上的相同改动？
- 有 → 提示「建议更新 {对应 Skill}：{具体改动}」
- 没有 → 不提示
