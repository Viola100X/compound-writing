# Personal OS：个人品牌操作系统

这是整个内容系统的地基。你的定位、经历、文风决定了后续所有环节的输出质量。

三个模块，每个都可以跳过，后续随时补充。

全部跳过也可以直接进入写作流程，但会提醒：
> 你还没有建立 Personal OS，当前使用通用风格。随时说「初始化」来建立。

---

## 工作流协议

### 交互风格

- 说人话，不要机械。像一个有经验的顾问在引导你，不是在填表
- 每个模块开始时，先说清楚「我们现在进入什么环节」「大概需要什么信息」
- 基于已有信息主动预填，让用户确认或修正，而不是从零开始问
- 用户可以一次性倒一大段信息，Claude 自动整理成结构化档案

### 进度追踪

每个模块完成后更新进度：

```
Personal OS 进度：
✅ 品牌定位  ⬜ 经历素材库  ⬜ 文风提取
```

### 链路引导

每个模块完成后，告诉用户下一步是什么，并给出选项：继续 / 跳过 / 退出。

全部完成后推荐下一个 skill（/topic 或 /inbox）。

### Telemetry

每次 skill 调用完成后，记录到 `skill-usage.jsonl`：


## 模块一：品牌定位

进入时说：

> **我们现在进入品牌定位环节。**
> 我需要了解三部分信息：你是谁、你的产品/项目、你的内容方向。
> 你可以一次性把这些发给我，也可以发产品介绍让我提炼。不确定的先跳过。

### 需要收集的信息

**1. 你是谁**
- 个人背景和当前角色
- 一句话定位（Claude 可以帮提炼，公式：[领域] + [身份] + [独特价值]）

**2. 你的产品/项目**
- 产品名称
- 所属行业（例：AI infra / DeFi / SaaS / 教育）
- 产品核心特点（解决什么问题、怎么解决）
- 产品愿景（这个产品最终要成为什么）
- 目标客户（卖给谁、他们的痛点）

**3. 内容方向**
- 目标读者是谁
- 主要发布平台（X / 公众号 / 小红书 / 其他）
- 写作目标（读者看完应该有什么感受或行动）
- 内容策略（如有）

### 处理逻辑

- 用户发来一大段信息 → Claude 自动整理成下方表格，请用户确认
- 用户发来产品介绍/官网/pitch deck → Claude 提炼后填表
- 用户只说了部分信息 → Claude 预填已知部分，标注缺失项，问要不要补

### 模块一完成 → 输出

整理成品牌档案表格：

```
✅ 品牌定位 完成

| 字段 | 内容 |
|------|------|
| 一句话定位 | [提炼结果] |
| 产品名称 | [名称] |
| 所属行业 | [行业] |
| 产品核心特点 | [一句话] |
| 产品愿景 | [愿景] |
| 目标客户 | [画像] |
| 内容策略 | [策略] |
| 主要平台 | [平台] |

有要改的直接说，没问题我们进入下一步：经历素材库。
```

## 模块二：经历素材库

进入时说：

> **我们现在进入经历素材库环节。**
> 我会用「英雄冒险之旅」12 个问题，帮你把人生经历变成可反复调用的内容素材。
> 每一步都是一个独立的内容选题来源。大概需要 15-30 分钟。
> 每个问题都可以跳过，后续随时补充。

### 引导方式

逐步提问，每步等用户回答后再进入下一步。用户说「跳过」则跳到下一步。

每步提问时，说清楚当前进度（第 X/12 步）和这个问题对内容创作的用处。

### 12 步完整清单

**第 1 步：普通世界**
你的出身起点。在哪里出生，原生家庭条件如何？
→ 用于：个人品牌故事、共鸣类内容

**第 2 步：冒险的召唤**
为什么会对现在从事的事业有想法？以前的过往经历中有过什么遭遇？
→ 用于：为什么做这件事类内容

**第 3 步：拒绝召唤**
一开始可能没有考虑过这个方向，按照世俗的期待选择了其他的方向？
→ 用于：真实感、去完美人设

**第 4 步：遇见导师/贵人**
什么样的机缘巧合下入行了，并且决定以此为自己的终身事业？
→ 用于：感恩类、行业入门类内容

**第 5 步：跨越第一道门槛**
进入这个领域后，都遇到过哪些困难？有没有想过放弃？最后又是怎么克服的？
→ 用于：创业早期复盘类内容

**第 6 步：测试、盟友、敌人**
在这个行业中，发现哪些人是可以合作的，哪些人是你要坚定远离的？
→ 用于：行业认知、人脉观点类内容

**第 7 步：通往最深处的洞穴**
你对这个行业本质的认知。
→ 用于：深度观点类、行业分析类内容

**第 8 步：考验**
从业经历中最大的一次挑战。
→ 用于：复盘类、经验分享类内容

**第 9 步：奖励**
熬过这次考验后，你收获了哪些？
→ 用于：成长叙事、价值观输出

**第 10 步：回头路**
回顾自己的从业经验，发现有了新的想去的方向，为什么想去？
→ 用于：转型叙事、新方向宣言

**第 11 步：带着灵药归来**
过往的从业经历，给你带来最有价值的三样东西。
例：过硬的专业实力（具体体现）/ 对客户需求更好的理解（新趋势下用户需要什么）/ 独特的方法论或资源
→ 用于：专业价值展示类内容

**第 12 步：新的世界**
接下来你的使命是什么？你的产品/业务现在处于什么阶段？
→ 用于：愿景类、招募类内容

### 12 步完成 → 自动提炼

#### 人生三部曲

Claude 自动从 12 步中提炼三个转折点：

```
第一幕：[起点] — 你从哪里来，最初的世界是什么样的
第二幕：[转折] — 什么事件彻底改变了你的方向
第三幕：[现在] — 你现在在做什么，为什么做这个
```

这三幕是你所有内容的叙事底层。每次写作都可以从中提取素材。

#### 素材标签化

Claude 自动为每个已完成步骤生成可调用标签：

| 步骤 | 素材标签 | 可用于 |
|------|---------|--------|
| 第 1 步 普通世界 | #出身 #起点 | 个人品牌故事、共鸣类内容 |
| 第 2 步 冒险召唤 | #初心 #动机 | 为什么做这件事类内容 |
| 第 4 步 遇见贵人 | #转折 #贵人 | 感恩类、行业入门类内容 |
| 第 5 步 跨越门槛 | #困难 #坚持 | 创业早期复盘类内容 |
| 第 7 步 行业本质 | #洞察 #认知 | 深度观点类、行业分析类内容 |
| 第 8 步 考验 | #挑战 #低谷 | 复盘类、经验分享类内容 |
| 第 11 步 灵药 | #方法论 #核心能力 | 专业价值展示类内容 |
| 第 12 步 新世界 | #使命 #愿景 | 愿景类、招募类内容 |

后续在选题和成稿环节，可以直接调用：「用我的第 8 步素材」「从我的三部曲第二幕切入」。

## 模块三：文风提取

进入时说：

> **我们现在进入文风提取环节。**
> 我需要 3-5 篇你觉得「最像自己」的已发布内容。长文、短推文、回复都行，数量比质量重要。
> 你可以直接粘贴内容，或者发文件路径/链接让我自己读。
> 也可以跳过这步，先用通用风格，后续再建立。

### 建立流程

**Step 1 — 用户提供语料**
- 3-5 篇已发布的、觉得「最像自己」的内容
- 格式不限：长文、短推文、回复均可
- 数量优先于质量：有 5 篇比精挑 1 篇更有效

**Step 2 — AI 自动分析**
Claude 基于语料完成以下工作（用户不参与）：
1. 提取思维模式：叙事弧线、论证偏好、世界观
2. 提取语言模式：句长分布、高频词汇、特有表达
3. 输出完整文风档案

**Step 3 — 用户确认**
把分析结果展示给用户，问：「按这个风格写出来，感觉对吗？」
- 对 → 直接使用
- 有偏差 → 只调整句式校准部分，思维模式不需要改
- 差很多 → 重新提供语料

### 文风档案结构

#### A. 思维模式（AI 自动生成，用户不填写）

- **叙事弧线**：[从哪出发 → 经历什么 → 到哪结束]
- **论证偏好**：[按比重排序：个人叙事 / 数字对比 / 对比反转 / 权威引用 / 情绪共鸣]
- **世界观**：[AI 提炼的核心认知框架，1-2 句]
- **雷达图**：真实性 / 权力 / 情感 / 逻辑 / 具体 / 行动（各 1-10 分）

#### B. 句式校准（AI 提取初稿，用户通过实际写作逐步修正）

- **句长分布**：[短/中/长句比例]
- **特有词汇**：[常用词——不需解释直接用]
- **口语强度**：[AI 容易过强、本人实际会降一档的词——对照表]
- **禁止清单**：[写完必须删的句型或词]

#### C. 真实语料样本

[3-5 个具体段落，代表「这就是我的感觉」]

### 持续迭代规则

每次完成一篇文章的改稿，Claude 检查：
- 这次改了哪些地方？
- 有没有出现 3 次以上的相同改动？

有规律 → 在对话末尾提示「建议更新文风 v1.x：[具体改动]」，用户确认后写入。

**核心原则**：
- 思维模式不需要用户审核细节，只需确认整体方向
- 句式校准是最容易被感知的层，也是最需要迭代的层
- 真实语料是整个文风档案最有价值的部分，要随时补充
