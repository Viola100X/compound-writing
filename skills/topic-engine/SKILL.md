# 爆款选题引擎 Skill

> **触发词**：「帮我找选题」「最近写什么」「这个选题行不行」「选题打分」「给我选题灵感」

---

## 两种模式

### 模式 A：选题生成（没想法时用）

用户说「帮我找选题」「最近写什么」，Claude 执行：

**Step 1：锁定选题来源**

从以下 7 个来源中，结合用户身份和近期热点，推荐 3-5 个选题方向：

| # | 来源 | 说明 | 爆款率 |
|---|------|------|--------|
| 1 | 实战复盘 | 你刚做过的事、踩过的坑、拿到的结果 | ⭐⭐⭐⭐⭐ |
| 2 | 认知差 | 你知道但大多数人不知道的事 | ⭐⭐⭐⭐⭐ |
| 3 | 热点借势 | 行业大事件 + 你的独特视角 | ⭐⭐⭐⭐ |
| 4 | 反常识 | 大众认为 A，但真相是 B | ⭐⭐⭐⭐ |
| 5 | 行业预判 | 你看到了别人没看到的趋势 | ⭐⭐⭐ |
| 6 | 方法论 | 你总结出的可复用流程/框架 | ⭐⭐⭐ |
| 7 | 素材库触发 | 从 knowledge-base 或 inbox 中找到的可写素材 | ⭐⭐⭐ |

**Step 2：生成选题卡片**

每个选题输出一张卡片：

```
## 选题：[一句话标题]

来源类型：[7种来源之一]
目标读者：[谁会关心这个]
核心冲突：[读者现有认知 vs 你要说的]
爆款潜力：[⭐ 1-5 星] + 一句话理由
推荐结构：[S1-S4 / A-D]
时效性：[常青 / 7天内 / 3天内]
```

**Step 3：用户选择后，写入 session.md 并自动衔接**

用户选中一个选题后：

1. **持久化**：把选中的选题写入 `.compound/session.md` 的「选题」段落（目录不存在先 `mkdir -p .compound`；已有 session.md 且是同一篇文章就更新，不是就覆盖新开一篇）。写入字段：标题、核心观点、目标读者、五维评分、推荐结构、时效性。同时更新 frontmatter 的 `current_phase: topic` 和 `article_id`。

2. **推荐下一步**（按中文触发词引导，不用斜杠命令）：
   - 素材充足 → 提示"说『帮我选结构』进入下一步"（content-structure）
   - 素材不足 → 提示"说『采访我』补素材"（ai-journalist）
   - 观点浅 → 提示"说『升华一下』让观点更深"（thought-elevation）

### 模式 B：选题评估（有想法时用）

用户说「这个选题行不行」「选题打分」，Claude 执行：

**五维评分（满分 50）**

| 维度 | 10分标准 | 1分标准 |
|------|---------|---------|
| **独特性** | 只有你能写，别人写不出这个视角 | 任何人都能写，没有差异化 |
| **冲突感** | 直接挑战主流认知，读者想反驳 | 正确的废话，没有张力 |
| **实用性** | 读完立刻能用，可操作 | 纯观点输出，没有抓手 |
| **时效性** | 正好踩中当下热点或行业节点 | 什么时候发都一样 |
| **传播性** | 读者看完想转发给朋友 | 看完就关了，不会分享 |

**评分输出格式**：

```
## 选题评估：[选题名]

独特性：[X]/10 — [一句话理由]
冲突感：[X]/10 — [一句话理由]
实用性：[X]/10 — [一句话理由]
时效性：[X]/10 — [一句话理由]
传播性：[X]/10 — [一句话理由]

总分：[X]/50
评级：[S级 ≥40 / A级 ≥35 / B级 ≥25 / C级 <25]

改进建议：
- [如何把这个选题从 B 级提升到 A 级]
```

**评级说明**：
- **S 级（≥40）**：立刻写，不要犹豫
- **A 级（≥35）**：值得写，可能需要补一个维度
- **B 级（≥25）**：有潜力，但需要找到更强的切入角度
- **C 级（<25）**：建议换方向，或等更好的时机

### 高频爆款公式（前 5）

| # | 公式 | 示例 |
|---|------|------|
| 1 | **我花了 X 做 Y，发现 Z** | 「我花了 3 个月测试 20 个 AI Agent，发现只有 3 个能真正落地」 |
| 2 | **大多数人以为 A，其实 B** | 「大多数人以为 AI 会取代程序员，其实它先取代的是产品经理」 |
| 3 | **X 件我希望早点知道的事** | 「做 Web3 创业前，5 件我希望早点知道的事」 |
| 4 | **为什么 X 注定失败/成功** | 「为什么 90% 的 AI Agent 创业注定失败」 |
| 5 | **从 X 到 Y 的真实路径** | 「从月薪 3 万到年入 500 万，我做对了这 3 件事」 |

## 与其他 Skill 的衔接

```
topic-engine（选题）
    ↓ 选中选题
content-structure（选结构）
    ↓ 确定骨架
ai-journalist / thought-elevation（深化）
    ↓ 补素材或升华
writing-style（出初稿）
    ↓ 打磨
title-generator（发布优化）
```

topic-engine 是整个创作流程的**起点**——在你还没有明确想法时，帮你找到值得写的方向。
