# 爆款选题模板库

> topic-engine 的选题公式参考库。按场景分类，直接套用。

---

## 一、实战复盘型（最高爆款率）

适合：你刚做完一件事，有真实数据或踩坑经验。

| # | 模板 | 变量说明 |
|---|------|---------|
| 1 | 我花了 [时间] 做 [事情]，发现 [反直觉结论] | 时间越具体越好，结论要出人意料 |
| 2 | [数字] 次失败后，我总结出 [方法论] | 数字要真实，方法论要可操作 |
| 3 | 做 [X] 第 [N] 年，说几个没人告诉你的真相 | N≥2 有说服力，真相要反常识 |
| 4 | 上周我 [具体动作]，结果 [意外结果] | 时间近=时效性强，结果要有戏剧性 |
| 5 | 我把 [X] 从 [差状态] 做到 [好状态] 的全过程 | 对比越大越好 |
| 6 | [X] 踩过最贵的一个坑：[具体描述] | 「最贵」制造好奇心 |
| 7 | 我测了 [N] 个 [产品/工具]，只推荐这 [M] 个 | N 大 M 小，筛选感 |
| 8 | 做 [X] 一年，我的 [关键指标] 从 [A] 到 [B] | 数据说话 |

## 二、认知差型（高传播率）

| # | 模板 | 变量说明 |
|---|------|---------|
| 9 | 大多数人以为 [常见认知]，其实 [真相] | 常见认知越普遍，反转越有力 |

| 11 | 为什么 [看起来对的事] 反而是错的 | 制造认知冲突 |
| 12 | [X] 行业不会告诉你的 [N] 件事 | 行业内幕感 |
| 13 | 你以为的 [X] vs 真实的 [X] | 对比制造落差 |
| 14 | [X] 最大的谎言：[具体谎言] | 「谎言」是强冲突词 |
| 15 | 关于 [X]，99% 的人搞反了 | 数字+反转 |
| 16 | [X] 不是 [大众理解]，本质上是 [你的定义] | 重新定义=认知升级 |

## 使用说明

1. 浏览模板，找到和你当前状态最匹配的类型
2. 填入你的真实素材（不要编造）
3. 用 topic-engine 的五维评分检验选题质量
4. 确认后进入 content-structure 选结构

**核心原则**：模板只是起点，真正的爆款来自你的真实经历和独特视角。模板帮你找到表达的角度，不是替代你的思考。
