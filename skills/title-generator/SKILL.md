# 标题生成器 Skill
> **触发词**：「生成标题」「帮我写 Hook」「推特开头怎么写」「小红书标题」「标题优化」

调用后：
1. 分析文章内容或核心观点
2. 判断内容类型（方法 / 故事 / 数字 / 对比 / 警告 / 名人）
3. 从 `library.md` 中选 2-3 个最匹配的模板
4. 每平台输出 3 个备选，附底层逻辑和模板编号

完整标题模板库见 `library.md`（Twitter 100 个 + 小红书 182 个）

---

## 使用方式

将文章（或「核心观点 + 目标读者」）粘贴进来，告诉我发哪个平台。

---

## 输出格式

```
## Twitter Hook（3 个备选）

**A**：[内容]
底层逻辑：[为什么有效] | 参考模板：[#编号]

**B**：[内容]
底层逻辑：[为什么有效] | 参考模板：[#编号]

**C**：[内容]
底层逻辑：[为什么有效] | 参考模板：[#编号]

推荐：A，因为 [理由]

---

## 小红书标题（3 个备选）

**A**：[内容]
封面文字：[≤10字]
底层逻辑：[为什么有效] | 参考模板：[#编号]

**B**：[内容]
封面文字：[≤10字]

**C**：[内容]
封面文字：[≤10字]

推荐：A，因为 [理由]
```

---

## 平台注意事项

**Twitter**：
- Hook = 第一句话，决定是否被展开
- 长文（Thread）用清单类或方法论类开头
- 短推用哲理类或反差类

**小红书**：
- 标题要含关键词（利于搜索流量）
- 情绪词要到位：小窍门 / 秘密 / 必看 / 千万别 / 绝了
- 配合封面文字使用（≤10 字，视觉优先）
